Libft - ft_strchr 游릭游댮

Evil tests made with 游땓游땓游땓 passion

Context : The strchr() function returns a pointer to the first occurrence
       of the character c in the string s.

Test material
	- read only string ex : "hello"
	- Original strchr (#include <string.h>) for comparison

Prototype
	char	*ft_strchr(const char *s, int c)

[游릭] Happy path
	ft_strchr("hello", 'l')
		-> "llo"
	ft_strchr("hello", 108)
		-> "llo"

[游릭] Modulo what??
	ft_strchr("hello", 108 + 256)
		-> "llo"
	ft_strchr("hello", 'l' + 16 * 256)
		-> "llo"

[游릭] Empty str
	ft_strchr("", 'l')
		-> Null

[游릭] Null address
	ft_strchr((char *)0, 32)
		-> Segfault

[游릭] Find the Terminator
	ft_strchr("hello", '\x0')
		-> ""
	ft_strchr("", '\x0')
		-> ""

[游릭] Nowhere to be found
	ft_strchr("hello", 'x')
		-> Null
	ft_strchr("hello", -2147483647)
		-> Null
	ft_strchr("hello", -1)
		-> Null
	ft_strchr("hello", 4242)
		-> Null
